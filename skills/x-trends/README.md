# X Trends Scraper 📉

Fetch current top trending topics on X (Twitter) for any country using public aggregators.

## Features

- Get trending topics for any country
- No X/Twitter account required
- Powered by [getdaytrends.com](https://getdaytrends.com)

## Installation

```bash
clawhub install x-trends
```

## Usage

Ask your agent:

- "What's trending on X in the US?"
- "X trending topics for Japan"
- "Get Twitter trends for Brazil"
