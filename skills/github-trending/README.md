# github-trending

获取 GitHub Trending 热门项目，支持按日/周/月筛选。

## 快速开始

```bash
github-trending            # 今日热门
github-trending weekly     # 本周热门
github-trending monthly    # 本月热门
github-trending json       # JSON 输出
```

## 输出

默认 Markdown 表格：排名、项目名、语言、今日 Stars、总 Stars、简介。

## 特点

- 无需 API Key，抓取公开页面
- 实时获取，无缓存
