## 组件说明

### 1. 顶部导航栏

- **标题**: "🎛️ Edge Agent Dashboard"
- **副标题**: "实时监控和管理边缘AI Agent"
- **连接状态指示器**:
  - 🟢 绿色 = 已连接
  - 🔴 红色 = 断开连接（带脉冲动画）
  - 🟡 黄色 = 连接中
- **"+ 添加Agent"按钮**: 打开添加Agent模态框

### 2. 资源监控卡片（4个）

#### CPU使用率卡片
- **数值**: 当前CPU使用率（百分比）
- **进度条**: 可视化显示使用率
- **颜色**: 蓝色

#### 内存使用卡片
- **数值**: 当前内存使用率（百分比）
- **进度条**: 可视化显示使用率
- **详细信息**: 已用/总内存（MB）
- **颜色**: 绿色

#### 网络发送卡片
- **数值**: 每秒发送流量（MB/s）
- **说明**: "最近1秒"

#### 网络接收卡片
- **数值**: 每秒接收流量（MB/s）
- **说明**: "最近1秒"

### 3. 实时图表（2个）

#### CPU & 内存趋势图
- **类型**: 折线图
- **数据**: CPU（蓝色线）和内存（绿色线）
- **X轴**: 时间（HH:mm:ss格式）
- **Y轴**: 百分比（0-100%）
- **更新**: 每秒更新，最多显示60个点
- **特性**: 无动画，高性能

#### 网络流量图
- **类型**: 折线图
- **数据**: 发送（橙色线）和接收（红色线）
- **X轴**: 时间（HH:mm:ss格式）
- **Y轴**: 流量（MB/s，自动缩放）
- **更新**: 每秒更新，最多显示60个点

### 4. Agents区域

#### 标题栏
- **标题**: "Agents"
- **计数**: "N 个Agent"

#### Agent卡片

每个Agent卡片包含：
- **Agent名称**: 显示名称
- **Agent ID**: 唯一标识符（灰色小字）
- **状态徽章**:
  - 🟢 [RUNNING] - 运行中
  - ⚪ [STOPPED] - 已停止
  - 🔴 [ERROR] - 错误
  - 🟡 [STARTING] - 启动中
  - 🟡 [STOPPING] - 停止中
- **详细信息**:
  - PID: 进程ID（运行时显示）
  - 运行时间: 格式化显示（如"2h 30m"）
- **错误信息**: 如果有错误，显示红色警告（单行截断）
- **操作按钮**:
  - **[停止]**: 停止Agent（红色，运行时显示）
  - **[重启]**: 重启Agent（黄色，运行时显示）
  - **[启动]**: 启动Agent（绿色，停止时显示）
  - **[日志]**: 查看日志（灰色）
  - **[🗑️]**: 删除Agent（灰色，悬停变红色）

**交互效果**:
- 卡片悬停：轻微上移 + 阴影增强
- 按钮悬停：颜色加深
- 状态徽章：根据状态不同颜色

### 5. 日志查看器

#### 标题栏
- **标题**: "日志查看器"
- **Agent选择器**: 下拉菜单选择Agent
- **[清空]按钮**: 清空当前日志

#### 日志容器
- **背景**: 深灰色 (#111827)
- **字体**: 等宽字体（Monaco/Menlo/Ubuntu Mono）
- **字号**: 12px
- **行高**: 1.4
- **滚动**: 垂直滚动（最大高度256px）
- **自动滚动**: 新日志到达时自动滚动到底部

#### 日志行
- **格式**: `[前缀] 日志内容`
- **前缀**:
  - `[OUT]` - 标准输出（绿色 #10b981）
  - `[ERR]` - 错误输出（红色 #ef4444）
  - `[INFO]` - 信息（继承颜色）
  - `[WARN]` - 警告（继承颜色）
  - `[DEBUG]` - 调试（继承颜色）
- **交互**: 悬停时背景变亮
- **换行**: 自动换行（word-break: break-all）

## 响应式设计

### 桌面（> 1024px）
- 4列资源卡片
- 2列图表
- 3列Agent卡片

### 平板（768px - 1024px）
- 2列资源卡片
- 1列图表
- 2列Agent卡片

### 手机（< 768px）
- 1列资源卡片
- 1列图表
- 1列Agent卡片

## 实际使用效果

### 初始状态
- 连接状态: 🟡 连接中...
- 资源卡片: 显示 "--%"
- 图表: 空白
- Agents: 显示"还没有Agent，点击'添加Agent'开始"
- 日志: 显示"选择一个Agent查看日志"

### 连接后
- 连接状态: 🟢 已连接
- 资源卡片: 显示实时数据
- 图表: 开始绘制
- Agents: 显示已配置的Agent
- 日志: 显示选中Agent的日志

### 运行中
- 资源卡片: 每秒更新
- 图表: 实时滚动
- Agent状态: 实时变化
- 日志: 流式显示（新日志自动滚动到底部）

## 交互体验

1. **平滑过渡**: 所有UI元素都有过渡动画
2. **即时反馈**: 按钮点击立即响应
3. **自动更新**: 无需刷新，数据自动更新
4. **容错设计**: 断线自动重连
5. **直观操作**: 图标和文字结合，易于理解
