## 🧪 Agent Memory Service 开发

### 新项目: agent-memory-service
- **位置**: `projects/agent-memory-service/`
- **状态**: ✅ v0.1.0 完成, 31/31 tests pass
- **特点**:
  - Mem0 风格三层存储: Core(L0,永不过期) / Long(L1,30天半衰期) / Short(L2,1天半衰期)
  - 自动记忆提取 Pipeline: 偏好/事实/决策/实体/上下文
  - 多策略检索: 关键词 + n-gram语义相似度 + 时间衰减 + 层级加权
  - Ebbinghaus 遗忘曲线衰减
  - 访问增强（recall即复习）
  - 内容哈希去重
  - JSON文件持久化
  - 零外部依赖

### 设计决策
- 不用 SQLite，用 JSON 文件 — 更简单，零依赖
- n-gram 相似度替代 embedding — 不需要 API，离线可用
- 规则提取替代 LLM 提取 — 快速原型，后续可升级
