## ✅ Created: Prompt Weaver 🧵

**Location**: `code-lab/prompt-weaver/`

### 项目概述

一个零依赖的轻量级 Prompt 编排引擎，用于快速原型开发。

### 核心特性

1. **模板引擎**
   - 变量替换: `{{name}}`
   - 过滤器: `{{name | upper}}`
   - 多变量支持

2. **链式 API**
   ```python
   Chain().prompt("...").transform("upper").output()
   ```

3. **条件分支**
   - 函数条件: `lambda ctx: ctx.get("value") > 5`
   - 字符串条件: `"{{score}} >= 60"`

4. **内置转换器**
   - upper, lower, trim
   - length, split, join
   - first, last, json

5. **YAML 配置**
   - 声明式工作流定义
   - 零依赖 YAML 解析

6. **Mermaid 可视化**
   - 自动生成流程图
