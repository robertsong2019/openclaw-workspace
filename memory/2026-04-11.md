## 代码实验室晚间实验

### 项目：Pocket Agent + Self-Evolving Agent
位置：`lab/pocket-agent/`

**Pocket Agent (~150行)**
- 最小化 AI Agent 框架，演示核心概念
- Tool Registry（自动从类型提示生成 JSON Schema）
- ReAct Loop（Reason → Act → Observe 循环）
- Episodic Memory（关键词匹配，可替换为向量搜索）
- MockLLM 可直接替换为 OpenAI/Anthropic API

**Self-Evolving Agent**
- Agent 从零工具开始，按需动态生成和注册工具
- 8 个内置工具模板：fibonacci, prime_check, password_gen, color_palette, text_stats, uuid_gen, base64_encode, hash_text
- 支持工具升级（generation 追踪）和代码检查
- 演示了 "Agent 可以在运行时扩展自身能力" 的概念

### 关键设计决策
- Mock LLM 而非真实 API → 零依赖，可离线运行
- 工具通过装饰器注册 → 类似 OpenAI function calling
- exec() 动态生成工具 → 概念验证自进化
