# 2026-04-05 - 开发工具

## ctxgen - AI 上下文文件生成器

今晚创建了一个实用的 CLI 工具：**ctxgen**

**位置：** `/root/.openclaw/workspace/code-lab/ctxgen/`

**功能：**
- 分析 Git 仓库结构、技术栈、代码模式
- 自动生成 AI 编码助手友好的上下文文件
- 支持多种目标：AGENTS.md、.cursorrules、CLAUDE.md、context.md

**使用：**
```bash
cd /root/.openclaw/workspace/code-lab/ctxgen
node cli.js <path>
node cli.js <path> --dry-run
node cli.js <path> --target agents.md
```

**技术栈：**
- 纯 Node.js，无依赖
- 模块化设计：analyzer.js、generator.js、writer.js
- 递归目录扫描，支持 .gitignore

**检测能力：**
- 语言识别（通过文件扩展名）
- 框架检测（Next.js、Vue、React、Express 等）
- 工具检测（Tailwind、Vite、Webpack、Docker 等）
- 代码规范识别（命名约定、测试模式、目录结构）

**验证：** ✅ 在 workspace 和 ctxgen 项目上测试通过
