### 3. 架构洞察：Heartbeat 的生命隐喻

**核心区别**:
- **Cron**: 基于时间，无状态，死板执行
- **Heartbeat**: 基于状态，有记忆，上下文感知，灵活调整

**自主 Agent 特质**:
1. **上下文感知** - 看当前在做什么，决定立即处理还是等待
2. **状态驱动** - 基于生命状态，有记忆
3. **灵活调整** - 随时调整优先级和任务清单
4. **自我调节** - 判断"现在不需要做什么" → HEARTBEAT_OK
