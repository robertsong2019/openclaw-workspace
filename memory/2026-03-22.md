# Code Lab Evening Session - 2026-03-22 21:00

## Project: Project Context Generator

**Goal**: Build a lightweight Python tool that analyzes codebases and generates AI-ready context summaries for quick onboarding of AI assistants.

## What I Built

### Core Tool: `context_gen.py`
A standalone Python script that:
- Scans project directories intelligently (ignoring common non-code directories)
- Detects programming languages based on file extensions
- Identifies dependency files (package.json, requirements.txt, Cargo.toml, etc.)
- Finds likely entry points (main.py, index.js, etc.)
- Extracts documentation previews
- Generates statistics (file counts, lines of code)
- Outputs in Markdown or JSON format

### Example Integration: `examples.py`
Demonstrates AI workflow integration with:
- Compact context generation (token-efficient one-liners)
- Full AI onboarding prompts
- JSON context for programmatic use

### Documentation: `README.md`
Comprehensive documentation with:
- Feature list
- Usage examples
- Use cases
- Roadmap

## Technical Highlights

1. **Smart Filtering**: Automatically ignores node_modules, .git, __pycache__, venv, etc.
2. **Language Detection**: Maps file extensions to human-readable language names
3. **Flexible Output**: Supports both Markdown (human-readable) and JSON (machine-readable)
4. **Zero Dependencies**: Pure Python 3, no external packages needed
5. **CLI-Friendly**: Simple command-line interface with argparse

## How to Use

```bash
# Basic usage
python3 context_gen.py /path/to/project

# JSON output
python3 context_gen.py /path/to/project -f json

# Save to file
python3 context_gen.py /path/to/project -o CONTEXT.md

# Compact context for AI
python3 examples.py /path/to/project
```
