### 🛠 Implementation

**Added Components:**

1. **autoMaintain(opts?) method** (new)
   ```typescript
   const result = await mem.svc.autoMaintain({
     threshold: 80,       // Score below this triggers maintenance (default 80)
     tasks: ['purge', 'consolidate', 'compactChangelog', 'decay', 'reindex'],
     dryRun: false       // If true, report without executing
   });
   // Returns: {
   //   triggered: true,  // Whether maintenance was needed
   //   score: 72,        // Current health score
   //   actions: { purge: 1, compactChangelog: {...} }  // Tasks executed
   // }
   ```

2. **Health-based task triggering:**
   - **purge**: Runs when `expiry` dimension < 90
   - **compactChangelog**: Runs when `changelog` dimension < 90
   - **consolidate**: Runs when `access` dimension < 70
   - **decay**: Always runs (if in task list)
   - **reindex**: Always runs (if in task list)

3. **Task whitelist:**
   - Configure exactly which tasks to run via `tasks` parameter
   - Default: `['purge', 'compactChangelog', 'decay', 'reindex']`

4. **Dry-run mode:**
   - `dryRun: true` returns recommendations instead of executing
   - Useful for previews and logging

**Use cases enabled:**
1. **Agent startup health check** — Auto-run maintenance on agent init
2. **Heartbeat-based maintenance** — Run autoMaintain() in periodic heartbeats
3. **Adaptive thresholds** — Different thresholds for different environments (dev vs prod)
4. **Preview before execution** — Use dryRun to log what would be done
5. **Selective maintenance** — Task whitelist for targeted cleanup

**Time savings:**
- Manual maintenance decision-making → automated health-based triggering
- No need to remember which tasks to run → autoMaintain figures it out
- Preview capabilities → avoid unexpected operations

**Integration example:**
```typescript
// Agent startup
const health = await mem.svc.autoMaintain({ threshold: 75 });
if (health.triggered) {
  console.log('Maintenance executed:', health.actions);
}

// Heartbeat / cron
setInterval(async () => {
  const result = await mem.svc.autoMaintain({ threshold: 80, dryRun: true });
  if (result.triggered && result.score < 60) {
    // Log low health state
    logger.warn(`Memory health degraded to ${result.score}:`, result.actions);
  }
}, 30 * 60 * 1000); // Every 30 minutes

// Preview mode
const preview = await mem.svc.autoMaintain({ dryRun: true });
console.log('Would run:', preview.actions);
```
