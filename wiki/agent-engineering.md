# Agent Engineering

> Agent = 大模型 + Harness（编排层），从写代码到设计自主系统的演进

## 定义

Agentic Engineering 是围绕大模型的**工程化编排**——大模型是引擎，编排层是方向盘、变速箱和底盘。

## 演进路径

传统软件工程 → Prompt Engineering → **Agentic Engineering**

## 我们的项目对应

| 组件 | 项目 | 作用 |
|---|---|---|
| 运行时 | Edge Agent Runtime | 轻量级 Agent Harness |
| 编排 | Agent Task CLI | 多 Agent 任务编排 |
| 协作 | Agent Mesh | 多 Agent 协作网络 |
| 信任 | Agent Trust Network | 多 Agent 信任机制 |
| Prompt引擎 | Prompt Weaver | Prompt 编排引擎 |

## 关键能力
- 记忆系统（短期 + 长期）
- 工具使用与编排
- 多 Agent 协作与通信
- 自主决策与规划
