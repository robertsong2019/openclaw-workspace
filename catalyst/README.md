# Catalyst

> Agent Mesh — 多Agent协调通信系统

## 🎯 概述

Catalyst 是一个Agent网格（Agent Mesh）系统，提供多Agent之间的协调和通信能力。支持预定义工作流，包括内容创作、设计、研究等场景。

## 📦 模块

- `components/agent_mesh.py` — 核心Agent网格实现
  - `AgentMesh` — 主协调器
  - `CONTENT_CREATION_WORKFLOW` — 内容创作工作流
  - `DESIGN_WORKFLOW` — 设计工作流
  - `RESEARCH_WORKFLOW` — 研究工作流

## 🚧 状态

早期开发阶段 — 目前包含核心Agent网格框架，工作流定义待完善。
