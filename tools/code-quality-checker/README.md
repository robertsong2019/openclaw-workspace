# Code Quality Checker (CQC)

一个强大的代码质量检查工具，统一检查代码风格、复杂度、安全等问题。

## 功能特性

- 🎯 **多维度检查**: 代码风格、复杂度、安全性、依赖管理
- 🔧 **灵活配置**: 支持自定义配置文件和检查规则
- 📊 **详细报告**: 提供JSON和控制台格式的详细报告
- 🚀 **易于使用**: 简单的CLI命令，快速上手

## 安装

```bash
npm install -g @openclaw/code-quality-checker
```

或者在项目中本地安装：

```bash
npm install code-quality-checker
```

## 使用方法

### 基本检查

```bash
# 检查当前目录
cqc check

# 检查指定目录
cqc check /path/to/project

# 运行所有检查
cqc check --all
```

### 指定检查类型

```bash
# 只运行ESLint检查
cqc check --eslint

# 检查代码复杂度
cqc check --complexity

# 检查安全问题
cqc check --security

# 检查依赖安全
cqc check --dependencies
```

### 输出格式

```bash
# 控制台输出（默认）
cqc check --format console

# JSON格式输出
cqc check --format json

# 保存结果到文件
cqc check --output report.json
```

### 初始化配置

```bash
# 创建默认配置文件
cqc init

# 强制覆盖现有配置
cqc init --force
```
