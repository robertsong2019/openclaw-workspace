# session-archiver Features

## Core (v1.0) ✅
- [x] Archive sessions to local JSON store
- [x] List archives (newest first, limit)
- [x] Full-text search across archives
- [x] Export: markdown, HTML, JSON
- [x] Statistics (count, disk usage, date range)
- [x] Clean old archives (dry-run support)
- [x] CLI interface (commander)
